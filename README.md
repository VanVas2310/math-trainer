# Math Trainer 

Простой и эффективный тренажер для развития навыков устного счета.

## О проекте

Math Trainer - это консольное приложение для тренировки математических навыков. Программа генерирует случайные примеры на сложение, вычитание и умножение, проверяет ваши ответы и показывает статистику прогресса.

**Для кого:** для школьников, студентов и всех, кто хочет улучшить навыки быстрого счета.

## Функциональность

- Генерация случайных примеров (+, -, ×)
- Проверка правильности ответов  
- Подсчет статистики успеваемости
- Простой и интуитивный интерфейс
- Возможность выхода в любой момент

## Требования

- Python 3.6 или выше
- Стандартные библиотеки Python (random)
